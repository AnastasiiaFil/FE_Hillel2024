CSS selectors overview

Selectors - patterns used to select the element(s) you want to style.
It is possible to divide CSS selectors into five categories:

  • Simple selectors (select elements based on name, id, class)
  • Combinator selectors (select elements based on a specific relationship between them)
  • Pseudo-class selectors (select elements based on a certain state)
  • Pseudo-elements selectors (select and style a part of an element)
  • Attribute selectors (select elements based on an attribute or attribute value)

On this page you can find the explanation of some CSS selectors from each category.

Description of some CSS selectors

The CSS element Selector

The element selector selects HTML elements based on the element name. The element selector was applied to the <h1> element on this page, centering its text content and setting a specific text color.

The CSS id Selector

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element. The id selector was applied to the <h2> element, centering its text content, adding a bottom border, and providing some padding at the bottom.

Note: An id name cannot start with a number.

The CSS class Selector

The class selector selects HTML elements with a specific class attribute. The class selector was applied to the <button> element (".accordion-btn"), and it styles the accordion buttons by setting the background color, text color, cursor, padding, and text alignment.

Note: A class name cannot start with a number.

The CSS multiple class Selector

The class selector selects HTML elements with a combination of a specific class attributes. The multiple class selector was applied to some <p> elements ("with-divider note-box"). The combination of these two classes ( .with-divider .note-box), applies styles that create a <p> element with a divider line between neighboring elements and a bordered box around the content.

Note: A class name cannot start with a number.

The CSS Universal Selector

The universal selector (*) selects all HTML elements on the page. The universal selector sets the font-family property to "Arial, sans-serif" for all elements in the HTML document.

Note: May be ineffective and may slow down the page rendering process, especially on pages with many elements.

A combinator is something that explains the relationship between the selectors.

The CSS Descendant Selector

The descendant selector matches all elements that are descendants of a specified element. The Descendant Selector on this page selects all <h3> elements that are descendants (located inside) of elements with the class .accordion-content and assigns them a specific color.

Child selector (>)

The child selector selects all elements that are the children of a specified element. Child selector (>): Setting padding for each element of <ul>, that is a direct descendant of the element with the class .accordion-content

Adjacent sibling selector (+)

The adjacent sibling selector is used to select an element that is directly after another specific element. Sibling elements must have the same parent element, and "adjacent" means "immediately following". Adjacent sibling selector (+): Set margin-bottom for each list item (li) inside the <ul> element, which is a descendant of the element with the class .accordion-content

General sibling selector (~)

The general sibling selector selects all elements that are next siblings of a specified element. General sibling selector (~): Apply style to each <ul> list element that immediately follows a <p> element.

A pseudo-class is used to define a special state of an element. For example, it can be used to:

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

The pseudo-class selector applies styles to elements with the class .accordion-btn when the mouse cursor is hovering over them (:hover pseudo-class). Thus, when a user hovers the cursor over the accordion button, the button's background changes to the specific color. In this case, the :hover pseudo-class allows for styling changes to the element upon interaction with it. The syntax of pseudo-classes:

                        
                            selector:pseudo-class {
                                property: value;
                            }

A CSS pseudo-element is used to style specified parts of an element. For example, it can be used to:

  • Style the first letter, or line, of an element
  • Insert content before, or after, the content of an element
                        
                            selector::pseudo-element {
                                property: value;
                            }

The [attribute] selector is used to select elements with a specified attribute.
